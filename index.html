Projects

Plagiarism Detection Project

A tool constructed in C++ that crosslists text in a file against a corpus of other files in order to detect direct plagiarism. This makes use of the n-gram approach in natural language processing and counts the number of matches that occur in each of the compared files. If this count exceeds a certain percentage threshold of the total text, then the compared document will be added to the set that will return all documents containing plagiarized material. The user has the option of specifying "n" (how big the sliding comparison window is) and the sensitivity level (what percentage of the total text must be similar in order to deem it plagiarized).

Blockchain Adversary Simulator

A simulator built in Java that aims to replicate how a majority miner (controls at least 51% of the hash power in a network) and a selfish miner (witholds solutions in the blockchain) would attack a public blockchain in order to gain an unfair advantage. Unit tests are included in order to verify that both miners are profiting from their respective techniques.
